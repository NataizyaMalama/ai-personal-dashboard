# AI Personal Dashboard

A personal productivity dashboard built with HTML, CSS, and JavaScript.

This project is being developed as part of my journey into full-stack and AI-powered web development. The goal is to create a dashboard that helps users track goals, manage tasks, and eventually receive personalized recommendations using AI.

## Current Features

* Add goals
* Delete goals
* Mark goals as completed
* Save goals using browser localStorage
* Press Enter to add goals
* Responsive dashboard layout
